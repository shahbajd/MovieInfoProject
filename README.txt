Movie Info Retrieval Project
October 2016 with ongoing revisions.
Copyright 2016 Shahbaj Dhillon

Description:

Main.py: 
Enter a movie title and program will search IMDb for the title. Program will also ask for a user rating. 
Proper title, year, and director will be retrieved. 
Information will then be written to a specific excel file whose location and name must be hardcoded into the software.

Mass Director Lookup:
Program will search through the existing excel file for titles with missing director fields. 
It will then one by one search up the title and retrieve the director and year and write it to the location in the excel file.
